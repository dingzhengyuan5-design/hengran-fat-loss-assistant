# 动作动画逐项审核规范

## 适用范围

本规范覆盖网站当前全部 220 项动作。任何动作必须先完成来源核对，再进入拟真人分镜生成；不得按动作模式批量套用相同步骤、提示或画面。

## 来源门槛

每个动作至少满足以下条件：

1. 至少两条相互独立的专业来源。
2. 其中至少一条来自医疗机构、专业认证机构、体育组织或开放专业动作库。
3. 至少一条来自能展示完整动作的视频或逐帧图文教程。
4. 博主内容只用于补充视觉轨迹、教学提示和常见错误，不能单独作为安全结论。
5. 不同来源存在合理差异时，记录为“个体化范围”，不得制造唯一精确角度。
6. 找不到足够来源时保持“待核对”，不得生成或上线。

优先来源包括：

- NASM、ACE、NHS、NSCA、ACSM及医院康复资料。
- E3 Rehab、Squat University、FitnessFAQs、Calisthenicmovement、Jeff Nippard、Renaissance Periodization等公开教学内容。
- 器械动作还必须核对器械厂商的调节、安全锁和轴线说明。

## 生成前审核

每项动作必须独立记录：

- 起始位和支点。
- 主要关节轨迹。
- 合理动作范围及可个体化部分。
- 回位方式和呼吸。
- 常见错误。
- 疼痛、眩晕、失衡、器械失效等停止条件。
- 适用的镜头角度、分镜布局和稳定锚点。

## 动画分型

| 动作形态 | 推荐布局 | 必须锁定的稳定锚点 |
| --- | --- | --- |
| 站立双脚固定 | 8×2，16 帧 | 双脚、地面、人物全局尺度 |
| 俯卧支撑 | 4×4，16 帧 | 手掌、脚尖或膝、地面 |
| 仰卧动作 | 8×2 或 4×4 | 肩胛/背部接触区、骨盆、器械 |
| 四点跪姿 | 4×4 | 支撑手、支撑膝、地面 |
| 单腿与箭步 | 8×2 | 前后脚或支撑脚、地面、骨盆朝向 |
| 悬垂动作 | 8×2 | 握点、横杠、躯干中心 |
| 器械动作 | 按器械比例选择 | 器械机架、转轴、座椅、脚踏 |
| 移动与有氧 | 按步态周期选择 | 地面、支撑脚切换、躯干中心 |

不得为了统一外观强制所有动作使用同一种分镜比例。

## 生成后五项复核

1. **解剖完整性**：肢体数量、关节连接、手指、鞋脚和器械接触正确。
2. **动作轨迹**：关节位移连续，动作顺序和合理活动范围符合审核记录。
3. **器械与安全**：握点、转轴、座椅、保护装置和负重路径正确。
4. **画面稳定**：镜头、背景、人物尺度和稳定支点无不合理漂移。
5. **说明一致性**：动画、步骤、常见错误和安全边界描述相互一致。

任一项不通过即进入返工队列。不得通过降低检查标准、隐藏问题帧或使用淡入淡出重影来放行。

## 上线门槛

- 来源审核、动作审核、动画复核和用户审核四个状态均通过。
- 审核清单记录核验日期、来源链接、版本号和已知个体化范围。
- 生成资产只使用自制拟真人画面；第三方视频仅作参考或依法嵌入，不复制其画面制作本站 GIF。
- 未通过的动作保留原有文字教程或显示“动画待审核”，不能伪装成已完成。
